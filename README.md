## 💡 What I've Learned So Far

- ✅ HTML document structure and boilerplate (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- ✅ Common tags and how attributes work (`href`, `src`, `alt`, `style`, `lang`)
- ✅ VS Code setup with Live Server + Emmet shortcuts
- ✅ Paragraphs, line breaks (`<br>`), horizontal rules (`<hr>`), preformatted text (`<pre>`)
- ✅ Inline CSS via the `style` attribute and Google Fonts integration
- ✅ Text formatting tags (`<b>`, `<strong>`, `<i>`, `<em>`, `<mark>`, `<del>`, `<sup>`, `<sub>`)
- ✅ HTML Quotations & Citations (`<blockquote>`, `<q>`, `<cite>`, `<abbr>`, `<address>`)
- ✅ Three ways to add CSS — Inline, Internal (`<style>`), External (`.css` file)
