Block Elements vs Inline Elements

What is a Block Element?

A block element always starts on a new line and stretches the full width of its container.

Example: <h1> and <p> are block elements

This H1 takes the full width (block)

This paragraph also takes the full width (block).

What is an Inline Element?

An inline element does NOT start on a new line. It only takes as much space as its content needs.

Example: <strong> and <a> are inline elements

Bold text (inline) and a link (inline) sit side by side in the same line.

Important Rule

Always place inline elements inside block elements — not the other way around.
